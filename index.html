California Infectious Disease Surveillance

The California Infectious Disease Surveillance Dashboard delivers a comprehensive analysis of a decade’s worth of public health data (2014–2023), built in Microsoft Power BI using Power Query for data transformation and DAX for incidence rate calculations to convert over 610,000 case records into a high-impact diagnostic tool for epidemiological trends. By visualizing the critical shift from raw case counts to population-adjusted incidence rates, the dashboard identifies regional hotspots such as the high-burden Kern County and reveals how environmental factors drive gender-specific disparities in diseases like Coccidioidomycosis. This project provides a sophisticated, data-driven roadmap for public health officials to optimize resource allocation and strengthen community intervention strategies across California.
